Spine.js - Executive summary

  • Created by Alex MacCaw (@maccman) while writing JavaScript Web Applications book
  • (Re)written in CoffeeScript.
  • Lightweight. Simple. 500 LOC. ~ 2K minified and gzipped
  • Inspired by Backbone.js. Also borrowed a few features from it
  • There is also Spine.mobile.js
  • For best experience you need to include jQuery
  • Has many plugins
  • JS compatible?
  • No views. Only controllers and models ... and classes
